# DSAIL-project

AI shopping engine (**Smouk**): style passport + conversational product discovery.

## App

The Next.js app lives in **`web/`**.

```bash
cd web
cp .env.example .env.local
# Add Supabase URL/key + OPENAI_API_KEY, run supabase/schema.sql in Supabase SQL editor
npm install
npm run dev
```

See **`web/README.md`** for full setup.
